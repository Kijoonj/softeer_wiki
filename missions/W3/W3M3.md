## MapReduce in Python

- What is Hadoop Streaming Jar?
    - .jar file is just a collection of .class files(compiled bytecode in java) packed together
    - Hadoop Straming Jar, located at..
        
        ```jsx
        /opt/hadoop/share/hadoop/mapreduce/hadoop-mapreduce-client-streaming-*.jar
        ```
        
        is an utility used as a universal language bridge, allowing us to use python or other languages to run the task.
        
    - It feeds stdin to the given script and catches stdout coming out from the MapReduce function.
- Where are Map, Reduce functions located? Do they have predefined location?
    - They do not have predefined location. They can be anywhere in the host container’s file system.
    - You map the MapReduce functions when running the task like below:
        
        ```jsx
        hadoop jar /opt/hadoop/share/hadoop/tools/lib/hadoop-streaming-*.jar \
          -files /opt/hadoop/src/mapper.py,/opt/hadoop/src/reducer.py \
          -mapper "python3 mapper.py" \
          -reducer "python3 reducer.py" \
          -input /input/local-test.txt \
          -output /output
        ```
